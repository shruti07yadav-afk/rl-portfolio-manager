# Multi-Asset Reinforcement Learning Portfolio Manager

## Overview

Developed a multi-asset reinforcement learning portfolio management system using PPO on ITC, HDFC, and TCS stock data.

## Results

| Metric | Value |
|----------|----------|
| Return | 5.90% |
| CAGR | 5.95% |
| Sharpe Ratio | 1.33 |
| Max Drawdown | -4.24% |
| Win Rate | 53.76% |
